FEATURES

Everything you need to
run code in the cloud

Monaco IDE Editor

Full-featured editor with syntax highlighting, auto-completion, line numbers, and bracket matching — the same engine powering VS Code.

Docker Sandboxing

Every execution runs in an isolated container with memory limits, CPU caps, no network access, and strict timeouts. Your server stays safe.

SHA-256 Caching

Identical code submissions are detected via hash and served from cache — skipping container spin-up entirely for instant results.

JWT Authentication

Secure signup and login with bcrypt-hashed passwords and JWT tokens. Protected routes keep your execution history private.

Execution History

Every run is stored in PostgreSQL. Browse your past submissions, rerun code, and track output across sessions with a searchable sidebar.

Rate Limiting

Per-user execution rate limits protect server resources. Configurable thresholds with graceful error responses keep the platform stable.
